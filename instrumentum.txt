The Instrumentum Project

Martin Heidegger, "Sein und Zeit", p. 74:
Zur Alltäglichkeit des In-der-Welt-seins gehören Modi des Besorgens, die das besorgte Seiende 
so begegnen lassen, daß dabei die Weltmäßigkeit des Innerweltlichen zum Vorschein kommt. Das 
nächstzuhandene Seiende kann im Besorgen als unverwendbar, als nicht zugerichtet für seine bestimmte 
Verwendung angetroffen werden. Werkzeug stellt sich als beschädigt heraus, das Material als ungeeignet. 
Zeug is hierbei in jedem Falle zuhanden. Was aber die Unverwendbarkeit entdeckt, ist nicht das 
hinsehende Feststellen von Eigenschaften, sondern die Umsicht des gebrauchenden Umgangs. In solchem 
Entdecken der Unverwendbarkeit fällt das Zeug auf. Das Auffallen gibt das zuhandene Zeug in einer 
gewissen Unzuhandenheit. Darin liegt aber: das Unbrauchbare liegt nur da -, es zeigt sich als Zeugding, 
das so und so aussieht und in seiner Zuhandenheit als so aussehendes ständig auch vorhanden war. Die 
pure Zuhandenheit meldet sich am Zeug, um sich jedoch wieder in die Zuhandenheit des Besorgtem, d.h. 
des in der Wiederinstandsetzung Befindlichen, zurückzuziehen. Diese Vorhandenheit des Unbrauchbaren 
entbehrt noch nicht schlechthin jeder Zuhandenheit, das so vorhandene Zeug ist noch nicht ein nur 
irgendwo vorkommendes Ding. Die Beschädigung des Zeugs ist noch nicht eine bloße Dingveränderung, ein 
lediglich vorkommender Wechsel von Eigenschaften an einem Vorhandenen. 

From "The Romance of Empire and the Politics of Self-Love", by T. de Zengotita (Harper's, July/2003):
Wielding things in general, wielding things in competition, wielding things in an emergency, in conflict. 
Primordial stuff. But intricate and artificial as well. The skill and discipline required, all the more 
so under pressure, never more so than in combat. The most violent ecstasis given form by technique. Exquisite 
balance at the outer limit. Is the urge for that balance irrepressible in us?

Skill with implements, hard or soft, is tantamount to fusion. One literally derives purpose from the tool. 
And the purposes inherent in tools fuse with those of other tools and, through them, with objectives - with 
the mission. [...]

To attend an implement with skill in the service of a mission is to belong, to be part of a greater whole, the 
aim of Romanticism fulfilled. 

Global Parameters: internode distance (real > 0)
                   number of molecules to create (integer > 0)
                   random number seed (integer > 0)
                   name of SQLite database file (string, file dialog for .sqlite3 file)
                   name of parameter file (string, file dialog for .txt file)
Pharmacophore Parameters: steric radius (real > 0)
                          number of pharmacophore points (integer > 0)
Carbon Skeleton Parameters: number of initial deletion attempts (integer > 0)
                            percentage deleted in initial phase (0 < real < 1)
                            start point for paths is random or pharmacophore point (radio button)
                            number of secondary deletion attempts (integer > 0)
                            number of quaternary carbons (integer >= 0)
                            number of four-ring carbons (integer >= 0)
                            number of rings (integer >= 0)
Rationalization Parameters: percentage of methyls to be eliminated (0 < real < 1)
                            minimum number of rings (integer >= 0)
                            maximum number of rings (integer >= min rings)  
Desaturation & Heteroatom Parameters: strip axial methyls (boolean) checkbox
                                      create aromatic rings (boolean) checkbox
                                      create double bonds (boolean) checkbox
                                      create triple bonds (boolean) checkbox (grayed out if double bonds not checked)
                                      create penta-atomic rings (boolean) checkbox
                                      create amides, esters and sulfonamides (boolean) checkbox (grayed out if double bonds not checked)
                                      substitute oxygen atoms (boolean) checkbox
                                      substitute nitrogen atoms (boolean) checkbox
                                      substitute sulfur atoms (boolean) checkbox
                                      substitute functional groups (phosphorus, fluorine, chlorine) (boolean) checkbox
Molecular Assembly Parameters: number of initial deletion iterations (integer > 0)
                               number of path selection iterations (integer > 0)
                               number of secondary deletion iterations (integer > 0) 
                               number of rationalization iterations (integer > 0)
                               number of desaturation & heteroatom iterations (integer > 0)

Three command buttons: Exit -> quit GUI
                       Write -> write current GUI parameters to parameter file if defined
                       Load -> if parameter file is defined and exists, load its values
